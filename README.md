# 🛠 SKpsblock — v1.0
**Підтримувана версія гри:** `1.16.5+` (Залежності: `WorldGuard`, `WorldEdit`)

---

### 📖 Про плагін
**SKpsblock** — це сучасна система захисту території за допомогою блоків-приватів (Protection Stones). Замість використання дерев'яної сокири та складних команд, гравці можуть просто встановити спеціальний блок, який автоматично створить захищений регіон навколо них.

Плагін інтегрований з **WorldGuard API** та використовує базу даних **SQLite** для відстеження стану кожного блоку привату, включаючи його унікальну механіку міцності.

---

### ✨ Ключові особливості

* **🏗 Автоматичний приват**
  Створення кубоїдного регіону в один клік. Достатньо поставити налаштований блок (наприклад, залізний), щоб миттєво захистити територію навколо нього.
* **💔 Система міцності (HP)**
  Унікальна механіка для PvP та Анархії: регіони мають запас здоров'я. При вибухах (TNT, кріпери) регіон не зникає миттєво, а втрачає HP. При 0 HP блок руйнується, а приват анулюється.
* **📊 Гнучкі ліміти**
  Підтримка дозволів на кількість регіонів (наприклад, `limit_ps_3`). Система автоматично перевіряє права гравця при спробі встановити новий блок.
* **🛡 Захист від накладання**
  Інтелектуальна перевірка перетинів: плагін не дозволить створити приват, який зазіхає на територію іншого гравця або вже існуючий регіон.
* **💾 Persistent SQLite Data**
  Вся інформація про власників, координати та HP блоків зберігається в локальній базі даних. Це гарантує стабільність навіть після перезавантаження сервера.

### 💻 Командний центр

| Команда | Опис | Доступ (Permission) |
| :--- | :--- | :--- |
| `/ps help` | Список усіх доступних команд | `ps.help` |
| `/ps info` | Інформація про регіон (HP, власник, учасники) | `ps.info` |
| `/ps add <player>` | Додати учасника до вашого привату | `ps.add` |
| `/ps remove <player>` | Видалити учасника з вашого привату | `ps.remove` |
| `/ps list` | Список ваших активних регіонів | `ps.list` |
| `/ps delete` | Видалити регіон з підтвердженням (10 сек) | `ps.delete` |
| `/ps reload` | Перезавантажити конфігурацію плагіна | `ps.reload` |
